# Abubkr Abbas — Portfolio & CV

Personal portfolio and CV site for **Abubkr Abbas Elkhider Mohammed**, AI Solutions Architect.

## Live Site

Deployed on [Render](https://render.com) as a static site.

## Structure

| File | Description |
|------|-------------|
| `index.html` | Portfolio landing page (homepage) |
| `cv.html` | Full professional CV |
| `landing.css` / `landing.js` | Landing page styles & interactions |
| `style.css` / `script.js` | CV page styles & interactions |
| `render.yaml` | Render deploy config (static site) |

## Local Development

Just open `index.html` in a browser, or run any static server:

```bash
npx serve .
```

## Deploy to Render

1. Push this repo to GitHub.
2. Go to [dashboard.render.com](https://dashboard.render.com) → **New +** → **Static Site**.
3. Connect your GitHub repo.
4. Render auto-detects `render.yaml` — click **Apply**.
5. Done. Your site is live on `https://abubkr-portfolio.onrender.com`.
